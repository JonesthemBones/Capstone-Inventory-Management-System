Dashboard Guide

To manage system users, assign roles, and control access permissions.


Overview

  • This page lets you view all users with their names, emails, roles (Admin, Manager, Staff), status, account creation dates, and last login activity. Use the search and filter tools to quickly locate users by name, email, role, or status.

Adding and Editing Users

  • Click the 'Add User' button to register a new user, filling out required details and assigning an appropriate role. Use the edit (pencil) icon to modify existing user accounts or update roles and details as needed.

Managing Users

  • Activate, deactivate, or remove users using the corresponding action buttons. Always review account status before restricting access to prevent disruptions to operations. The table also provides a quick view of each user's most recent activity.

⚠️Regularly review the user list for inactive and unauthorized accounts and update roels as staff responsibilities change.⚠️
